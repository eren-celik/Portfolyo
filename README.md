# Portfolyo

The portfolio is a cryptocurrency, currency, and metal(Gold) price tracker, in addition, the app including in different category the news section.


In inside
• VIPER architecture
• Sectioned TableView
• Generic VC Builder
• Paging View
• Search
• Moya - Network Layer
• Charts(data source is not ready)
